# BMW Used Car Market Dashboard

This project presents an interactive dashboard exploring the BMW used car market using a public Kaggle dataset.
（Click on the "bmw-dashboard" link to view the visual analysis.）

## Contents
- Interactive flexdashboard HTML
- Source code in R Markdown
- Dataset description
- Interactive widgets for price, mileage, and fuel type comparison

## Data Source
Used Car Dataset: Ford and Mercedes  
Kaggle: https://www.kaggle.com/datasets/adityadesai13/used-car-dataset-ford-and-mercedes

## Why this project matters
This project matters because used-car buyers often need simple tools to compare price patterns across many listings.  
The dashboard makes those patterns easier to understand and supports more informed purchasing decisions.
